R.js - i18n for JavaScript

Browser

In your HTML

<script type="text/javascript" src="my/project/R.js"></script>
<script type="text/javascript">
    R.registerLocale('en-GB', {
        // English (Source)    // Translated String
        'A Translated String': 'A Translated String',
    });
    R.registerLocale('fr', {
        // English (Source)    // Translated String
        'A Translated String': 'Une chaîne traduite',
    });
</script>
<script type="text/javascript">
    var h1 = document.createElement('h1');
    h1.innerHTML = R('A Translated String'); 
</script>

Node.JS

In your App
var R = require('R.js').R;
var express = require('express');
var app = express.createServer();

R.registerLocale('en-GB', {
    // English (Source)    // Translated String
    'A Translated String': 'A Translated String',
});
R.registerLocale('fr', {
    // English (Source)    // Translated String
    'A Translated String': 'Une chaîne traduite',
});

app.get('/', function(req, res) {
    response.send('<h1>'
        + R('A Translated String')
        + '</h1>'
    );
});

What?

R.js is a simple, lightweight (less than 1kb), internationalisation library for JavaScript, for the browser, or server. R.js has no dependencies, it works well with any other libraries you wish to use, such as jQuery.

On the browser, it offers a way of getting i18n on your JavaScript layer, which is great for frontend-ony apps, such as a Chrome App or heavy JavaScript app. It even has a special support bridge for ender to be used as an ender module (`ender build R.js`).

On the server, thanks to the CommonJS compatibility, it becomes an easily includable versatile i18n tool for any application you have running on NodeJS or similar.
